This POC demonstrates CSS Modules with a basic React setup.

  • Using https://github.com/css-modules/css-modulesify.
  • Unfortunately, https://www.npmjs.com/package/babel-plugin-react-css-modules did not work as documented.
  • ES6 transpiling & module bundling are managed as NPM scripts inside "package.json" using Browserify / Babelify. No need for Webpack or Gulp.
